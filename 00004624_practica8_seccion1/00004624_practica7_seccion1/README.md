¿Cuál es la diferencia entre autenticación y autorizacion?
Autenticación verifica quién eres. Es el proceso de confirmar tu identidad (por ejemplo, con usuario y contraseña) y autorización verifica qué puedes hacer. Una vez autenticado, determina qué recursos o acciones tienes permitido usar.
¿Cuál es la función del token JWT en la guía?
El token JWT (JSON Web Token) se usa para mantener la sesión del usuario una vez autenticado. Contiene información codificada (como el ID de usuario y permisos) que el servidor puede verificar sin necesidad de consultar la base de datos constantemente.
